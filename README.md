# ZJU盲审结果监控工具

本程序用于自动监控浙江大学研究生盲审结果，当结果更新时自动发送钉钉通知。

## 项目说明
- 基于开源项目使用AI进行修改：https://gist.github.com/FanBB2333/229d177bbffdb1adc96f5f8a65a3c47f
- 支持 **Windows 系统 EXE 直接运行** 
- 采用无头浏览器后台运行，不弹出窗口
- 支持账号、密码、钉钉 Webhook 本地保存

## 功能
- 自动登录浙大统一认证平台
- 自动保存登录状态（免重复输入密码）
- 每隔 60 秒自动刷新盲审页面
- 盲审结果**更新时**自动发送钉钉通知
- 后台稳定运行，崩溃自动重启

## 使用前准备（重要）
1. 创建一个钉钉群（可先拉一位好友再踢出，群可保留）
2. 按照教程创建钉钉机器人：
   https://gist.github.com/FanBB2333/229d177bbffdb1adc96f5f8a65a3c47f
3. **安全设置 → 自定义关键词 → 添加关键词：盲审**
4. 复制机器人的 Webhook 地址

## 使用方法
1. 运行程序
2. 输入 **浙大统一认证账号、密码**
3. 输入 **钉钉机器人 Webhook**
4. 程序开始后台监控，有更新会自动发钉钉通知

## 本地文件说明（保存在 EXE 同目录）
- `cookies.pkl`：登录状态（自动登录用）
- `ding_config.json`：钉钉 Webhook 配置
- `last_result.json`：上次盲审结果缓存

以上文件**仅保存在本地**，不会上传到任何服务器。

## 注意事项
- 程序仅本地运行，不会收集任何信息
- 账号、密码、Webhook 均只存在本地
- 如需修改信息，删除上述 **3 个文件**重新运行即可
